# School-library

> This is a school library project implemented on Ruby

### Requirements
- Ruby installed

### Clone this repository

```

$ git clone https://github.com/vigehi/library.git
$ cd library

```

### Installations

- On the root directory, run 'bundle install' to install the rerequired dependencies
